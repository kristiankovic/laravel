php artisan migrate -> Ejecuta todas las migraciones pendientes. Crea las tablas en la base de datos.

php artisan migrate:rollback -> Elimina las tablas creadas en la última tanda de migraciones ejecutando el método down.

php artisan make:migration create_*_table -> Crea una nueva migración con la estructura de la clase ya hecha.

php artisan migrate:refresh -> Empieza a ejecutar todos los métodos down de todas las migraciones y luego las vuelve a crear

php artisan migrate:fresh -> Elimina todas las tablas que hay en la base y luego las vuelve a crear

usar refresh o fresh cuando todos los datos han sido insertados con
migraciones

refresh -> solo elimina las tablas creadas con las migraciones y no las
que se crean de manera manual pero borra los datos de las tablas ingresados manualmente
